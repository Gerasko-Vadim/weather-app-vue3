# Local Weather
## Как это работает?
#### Можно осуществлять поиск городов введя в поисковую строку нащвание города и кликнуть на город из выпадающего списка
#### Сохранения города, кликнув на плюс в панели навигации
#### Смена темы, дневная - ночная
#### Смена языка 
#### Удаления города
